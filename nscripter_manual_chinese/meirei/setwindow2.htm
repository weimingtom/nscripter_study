setwindow2

setwindow2 "window cg file name"
-or-
setwindow2 #rrggbb

（EX） setwindow2 ":c;twindow2.bmp"

This modifies the appearance of the text window without deleting either it or the text log.
This command can only modify the image file or the color that was first specified in setwindow, and none of the other parameters.

setwindow2

setwindow2 "窗口cg文件名"
-or-
setwindow2 #rrggbb

（例） setwindow2 ":c;twindow2.bmp"

这个命令修改文本窗口的外观，而不删除它或文本日志。
这个命令只可以修改第一次指定在setwindow中的图像文件或颜色，不能改变其他参数。

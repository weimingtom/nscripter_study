menuselectvoice

menuselectvoice <character string>,<character string>,<character string>,<character string>,<character string>,<character string>,<character string>

Designates the WAV files that are played for menu choices, and other system sounds.

From left to right, the strings are: menu open, menu cancel, menu item mouseover, on click, forbidden operation, "yes" confirmation, "no" confirmation.

If any of these are set to "", no sound plays.
By default, all of these sounds are set to "".

menuselectvoice

menuselectvoice <文字列>,<文字列>,<文字列>,<文字列>,<文字列>,<文字列>,<文字列>

メニュー選択肢の音、その他システム音WAVEファイルの指定です。
順に、メニューが開くとき、メニューをキャンセルしたとき、 メニュー選択肢のどれかの上に動いたとき、どれかをクリックしたとき、 できない操作をしたときの警告、「はい」を選んだとき、「いいえ」をえらんだとき、です。

なお、WAVEファイルに""を指定すると、何も鳴らさなくなります。
デフォルトは、すべて""です。

menuselectvoice

menuselectvoice <字符串>,<字符串>,<字符串>,<字符串>,<字符串>,<字符串>,<字符串>

指定菜单选择项的声音、其他系统音wave文件。
依次是，菜单打开时，取消菜单时， 在某个菜单选择项上方移动时，点击某个菜单时， 不能操作时的警告，选择「是」时，选择「否」时的声音

而且，如果WAVE文件指定为""，不会让任何东西发声。 默认，全部是""。

mode_saya

A command mode written especially for the doujin game, 「Saya～Labyrinth of Immorality～」. Once you specify this command, you enter: the legendary Saya Mode!
In Saya mode:

  • When the text window disappears, the sprites 0-9 also disappear, regardless of whether they're flagged to be visible or not.
  • When the text window disappears, the bars and numbers indicating status underneath the text window also disappear.
You would do these two things, for instance, when you had a parameter status window that you wanted to hide when the text window disappeared.

mode_saya

一个命令模式，专用于同人游戏，「Saya～Labyrinth of Immorality～」。 一旦你指定这个命令，你进入传说中的沙耶模式！
在沙耶模式中：

  • 当文字窗口消失时，精灵0-9也消失，不管它们是否被标记为可见。
  • 当文字窗口消失时，在文字窗口下方指示状态的条块和数字也消失。
你将做这两件事，例如，当你有一个参数状态窗口，当文字窗口消失时你希望隐藏它。

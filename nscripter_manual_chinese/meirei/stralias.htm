stralias

stralias <character string 1>,<character string 2>

Makes an alias of character string 2 as character string 1 -- in other words, after the declaration you can refer to character string 2 as character string 1 in all cases.

　　　　EX）
　　　　stralias man_a0 ":a/10,20,2;man_alpha.bmp"

Now, instead of typing in ":a/10,20,2;man_alpha.bmp", you can type in man_a0.
For instance, now ld c,man_a0,1 is equivalent to ld c,":a/10,20,2;man_alpha.bmp",1.

stralias

stralias <文字列1>,<文字列2>

ファイル名エイリアスを作ります。以降、ファイル名の代わりに
そのエイリアスを使って指定できるようになります。

　　　　例）
　　　　stralias man_a0 ":a/10,20,2;man_alpha.bmp"

以後、このファイルを指定したいときにはman_a0というエイリアスを使える。
たとえば、ld c,man_a0,1　は、ld c,":a/10,20,2;man_alpha.bmp",1　と同じ。

stralias

stralias <字符串1>,<字符串2>

制作文件名的别名。以后用别名代替文件名使用

　　　　例）
　　　　stralias man_a0 ":a/10,20,2;man_alpha.bmp"

在以后，使用man_a0代替":a/10,20,2;man_alpha.bmp"。
例如现在ld c,man_a0,1等效于ld c,":a/10,20,2;man_alpha.bmp",1。

puttext

puttext <character string>

Indicates the designated character string onscreen. This command is most often used within if statements to display some short indicator, but otherwise its efficacy is limited.
When using puttext, you may not use any of the variable designation/value functions (% or $, for instance) with it.

puttext

puttext <文字列>

その文字列を表示します。if文のあとで、ちょっとした表示をやりたいとき などは使えるかもしれません。
puttextのときは、変数を表示する機能 （特殊表示文字%と$）は使えません。

puttext

puttext <字符串>

显示指定的字符串。 这个命令可能用于在if语句之后，想要做一些常用的显示的时候。
使用puttext时、不能使用显示变量的功能（特殊显示文字%和$）。

exbtn_d

exbtn_d <control string>

Determines the state of the composite button when the mouse cursor is on no part of it. In order to see the format of the control string, pease see the documentation for exbtn.
　　EX）exbtn_d "P3C4C5C6P7,0"

exbtn_d

exbtn_d <控制字符串>

决定组合按钮的状态，当鼠标光标不在它的上面。 要想查看控制字符串的格式，请参考exbtn的文档。
　　例子）exbtn_d "P3C4C5C6P7,0"

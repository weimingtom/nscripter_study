btntime

btntime <numerical variable>

This designates a time limit in milliseconds for the next btnwait or btnwait2 period. Once this command is used and btnwait is activated, and if nothing happens in the specified time period, then btnwait will return -2.

btntime

btntime <数字变量>

此命令指定一个与下一个btnwait或btnwait2之间毫秒单位的时间限制。如果此命令使用并且btnwait被激活，在指定的时间间隔内没有事情发生，btnwait将返回-2。

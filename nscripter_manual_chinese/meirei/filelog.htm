filelog

This allows for creation of a file log.
Every single resource used by the program becomes recorded in that file log as a tag.
If you specify neither this command nor globalon, then nscr.exe will not generate anything, and will thus be allowed to run from CD.

filelog

filelog

ファイルアクセスログをとることを指定します。
一度でも読み込んだ画像タグは、すべて記憶します。
なお、タグが違えば、同じファイルでも違うものとみなされます。
この命令もglobalonも使わなければ、NScr.exeはカレントに 何も書きに行きませんので、ＣＤ起動が可能になります。

filelog

filelog

指定获取文件访问日志。
每当读取图片标签时，全部都记录下来。
而且，如果标签不相同，即便文件相同也会当作不同的东西。
如果既不使用这个命令也不使用globalon，当前NScr.exe将不会写入任何东西，以允许CD启动。

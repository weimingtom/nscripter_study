versionstr

versionstr <character string 1>,<character string 2>

Designates version strings to display in the about dialog. The first string is the first line, the second string is the second line.

versionstr

versionstr <文字列1>,<文字列2>

バージョン情報のダイアログに表示する文字列を変えます。
左が一行目、右が二行目です。

versionstr

versionstr <字符串1>,<字符串2>

改变版本信息的对话框中显示字符串。
左边是第一行，右边是第二行。

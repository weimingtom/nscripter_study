spbtn

spbtn <sprite number>,<button number>

Just like the btn command, except it sets up an image button using a sprite. The graphic for the nonclicked state goes in cell 0, the graphic for the clicked state goes in cell 1. Once the button is pressed, it returns the button number as a value.

spbtn

spbtn <精灵编号>,<按钮编号>

类似btn命令，不同的是它使用精灵配置图像按钮。 用于非点击状态的图像在格子0， 用于点击状态的图像在格子1。 一旦按钮被按下， 它返回按钮编号作为一个值。

spi

spi <character string>

Allows for utilization of image compression plugin dll's -- thus for image filetypes that are not supported in stock nscripter. Usage for the character string goes "plugin dll name|file extension".

　　　EX）spi "jpgplgin.dll|jpg"
　　　　　spi "iflf2.spi|lf2"

spi

spi <文字列>

画像圧縮プラグインを指定します。書式は、"ファイル名|関連付け拡張子"です。

jpegについては指定の必要はありません。

spi

spi <字符串>

指定图像压缩插件。语法是，“文件名|关联后缀名”。

对于jpeg后缀没有必要指定。

dwaveload

dwaveload <number>,<filename>


Loads a WAV file into memory for use with the commands dwaveplay and dwaveplayloop.

EX）
dwaveload 0,"test.wav"
dwaveplay 0 dwaveplayloop 0

dwaveload

dwaveload <编号>,<文件名>


加载一个wav文件进内存，用于使用命令dwaveplay和dwaveplayloop。

例子）
dwaveload 0,"test.wav"
dwaveplay 0 dwaveplayloop 0

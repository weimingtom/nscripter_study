clickvoice

clickvoice <character string>,<character string>

The first string is the name of the WAV file played when a button is clicked in the click wait state. The second string determines whether this is a default click wait state, or a form-fed click wait state.

clickvoice

clickvoice <文字列>,<文字列>

クリック待ちでクリックしたときになるWAVEファイルを指定します。
順に、普通のクリック待ち、改ページクリック待ち、です。

clickvoice

clickvoice <字符串>,<字符串>

指定等待点击时点击声音的wave文件。
参数依次是普通等待点击声和换页点击声。

selnum

selnum <numerical variable>,<character string>,<character string>,…

Works somewhat like select and selgosub, except for the fact that depending on which character string is clicked, a number (starts at 0, then increments by one for each string) is loaded into the numerical variable instead of any kind of jump happening.

selnum

selnum <数字変数>,<文字列>,<文字列>,…

選択肢の結果を、数字変数に代入します。一番左が０で、以降１ずつ増えます。

selnum

selnum <数字变量>,<字符串>,<字符串>,…

选择项的结果，代入到数字变量中。最左开始为0，以后逐个加1。

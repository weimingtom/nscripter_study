loadgame

loadgame <save number>

　Loads the save data corresponding to the save number. As it does not prompt for confirmation, be careful.
　This command gives no error indication if save cannot be loaded, and proceeds to the next command without stopping.

loadgame

loadgame <存档数字>

加载存档数字对应的保存数据。但是它不提示确认，请小心。
如果存档不能被加载，这个命令不给出错误指示， 而且不会停止地前进到下一条命令。

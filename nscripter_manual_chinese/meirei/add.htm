add

add <numerical variable>,<number>

Add two numbers. If used on characters, causes string concatenation instead.

　　　　EX）
　　　　add %0,6
　　　　add $5,"aaaa!"

add

add <変数名>,<値>

変数に変数を加えます。文字の場合は連結です。

　　　　例）
　　　　add %0,6
　　　　add $5,"ああああ"

add

add <变量名>,<数字或字符串>

数字型变量的值增加。字符串变量则为连接。

　　　　例）
　　　　add %0,6
　　　　add $5,"啊啊啊啊"

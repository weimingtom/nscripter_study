humanz

humanz <number>

Designates the image priority when sprites and pictures overlap (the z-order).
By using this command, you can cause pictures to be drawn directly on top of sprites.
The default value for this is 25.

humanz

humanz <数字>

スプライトと立ち絵の重なる時の優先順位（Ｚオーダー）を指定します。
指定した番号のスプライトのすぐ上に立ち絵が表示されることになります。
デフォルトは、25です。

humanz

humanz <数字>

精灵和立绘重叠时的优先顺序（Z序）。
可在指定编号的精灵上显示立绘。
默认为25。

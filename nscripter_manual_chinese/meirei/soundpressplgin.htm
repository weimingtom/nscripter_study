soundpressplgin

soundpressplgin <character string>

Using this, you can designate a dll file with which to decode other kinds of compressed sound files that are not mp3. The usage for the character string is "plugin name|3-character file extension".

　　　EX）soundpressplgin "nbzplgin.dll|nbz"

soundpressplgin

soundpressplgin <文字列>

音楽圧縮プラグインを指定します。書式は、"ファイル名|関連付け拡張子"です。

　　　例）soundpressplgin "nbzplgin.dll|nbz"

soundpressplgin

soundpressplgin <字符串>

指定音乐压缩插件。语法是“文件名|关联后缀名”。

　　　例）soundpressplgin "nbzplgin.dll|nbz"

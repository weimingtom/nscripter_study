print <effect>

Draws to the screen utilizing designated effect.

You may designate the effect to use in one of two ways:
１）By effect tag
　　　　0…draws only to memory, not to screen
　　　　1…instantaneous display
　　　　>1…the tag of an effect defined by the "effect" command
　　　　EX）bg "test.jpg",0
　　　　ld c,"testman.jpg",1
　　　　cl c,3

２）Direct designation
　　　　Use the same parameters as the "effect" command to directly designate an effect
　　　　EX）bg "test.jpg",10,1000
　　　　ld "testman.jpg",15,2000,"mask.bmp"

print

print <エフェクト>

そのエフェクトで画面を書きなおします。

エフェクトの指定には、次の二種類の方法があります。
１）一ケタで指定
　　　　0…何もしないで、メモリにだけ書きこむ
　　　　1…瞬時表示
　　　　その他…effect命令で定義済みのエフェクトを呼び出す。
　　　　例）bg "test.jpg",0
　　　　ld c,"testman.jpg",1
　　　　cl c,3

２）直接指定
　　　　effect命令に与えるパラメータと同じ物を指定。
　　　　例）bg "test.jpg",10,1000
　　　　ld "testman.jpg",15,2000,"mask.bmp"

print

print <特效>

用指定特效重新绘画屏幕。

对于指定特效，有以下两种方法。
１）一位数指定
　　　　0…什么都不做，只是写入到内存中
　　　　1…立即显示
　　　　其他值…调用已经通过effect命令定义了的特效。
　　　　例）bg "test.jpg",0
　　　　ld c,"testman.jpg",1
　　　　cl c,3

2)直接指定
　　　　指定与传给effect命令的参数相同的东西。
　　　　例）bg "test.jpg",10,1000
　　　　ld "testman.jpg",15,2000,"mask.bmp"

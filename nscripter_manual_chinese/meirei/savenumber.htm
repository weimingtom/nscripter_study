savenumber

savenumber <number>

Change the limit on the number of save files. At most, this can be 20, and the default is 9.
If you are not careful to make sure that the save menu does not overflow from the limits of the screen (using menusetwindow), this command can cause the interpreter to hard crash, so be careful.

savenumber

savenumber <数字>

セーブの最大数を変更します。最大２０までです。デフォルトは９。
menusetwindowで、画面からセーブメニューがはみ出ないような設定にしないと、 メニューを出した瞬間落ちるので注意してください。

savenumber

savenumber <数字>

更改存档的最大数。最大直至20。默认为9.
如果不用menusetwindow设置从屏幕中不要显示存档菜单的话， 菜单显示的瞬间会崩溃。请注意。

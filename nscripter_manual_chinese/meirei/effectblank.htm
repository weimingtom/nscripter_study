effectblank

effectblank <number>

This simply sets (in milliseconds) how long the interpreter should wait after the effect is over before interpreting the next command.
If the effect happens to have been built-in effect index 1, then this command has no effect, and there thus is no delay.

effectblank

effectblank <数字>

エフェクト終了後、次の命令の実行までにどれだけの間を取るかを ミリ秒単位で指定します。
エフェクトが瞬時表示(1)である場合、この設定に関係なく、 瞬時に表示されます。

effectblank

effectblank <数字>

效果结束后，保留多少时间才再执行下一个命令。
效果结束后，用毫秒单位指定到下一个命令执行的时间。
效果在瞬间显示（1）的情况下，瞬间显示，与这个设置无关。
